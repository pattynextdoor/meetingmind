## What the template covers

- Frontmatter — title, description, dates, tags, FAQ schema, HowTo schema
- Auto-generated table of contents from `##` headings
- Author byline, reading time, last-updated date
- Syntax-highlighted code blocks via Shiki
- Embedded images with width/height (no CLS)
- Internal links to other site pages

- A plain-markdown variant at `/blog/example-post.md`
- FAQ block + `FAQPage` schema
- HowTo block + `HowTo` schema

## How to author a new post

1. Copy this file to `content/src/content/blog/your-slug.mdx`.
2. Fill in the frontmatter — keep `description` under 170 characters.
3. Write your H2 sections. Each becomes a TOC entry automatically.
4. Optionally fill `faq` or `howto` frontmatter to emit rich-result schema.
5. Run `npm run check:seo && npm run check:schema && npm run check:llms` before merging.

## Code block example

```typescript
// Auto-link mention to existing vault note
function autoLink(mention: string, vault: VaultIndex): string {
  const match = vault.findByAlias(mention);
  return match ? `[[${match.title}]]` : mention;
}
```
