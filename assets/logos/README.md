# MeetingMind Logos

This directory contains the MeetingMind logo files in PNG format, optimized for both light and dark backgrounds.

## Files

- `meetingmind_logo_240.png` - 240x240px logo for smaller displays (navbars, favicons, etc.)
- `meetingmind_logo_600.png` - 600x600px logo for larger displays (hero sections, social media, etc.)

## Usage

These logos work well on both light and dark backgrounds. Use the appropriate size based on your display context:

- **240px**: Navigation bars, favicons, small badges
- **600px**: Hero sections, social media previews, large displays

## Locations

Copies of these logos are also available in:
- `docs/public/` - For documentation site
- `frontend/public/` - For landing page
