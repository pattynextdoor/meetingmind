# Getting Started

MeetingMind transforms your meeting transcripts into connected, actionable notes in Obsidian.

## What is MeetingMind?

MeetingMind is an Obsidian plugin that:

- **Imports transcripts** from multiple sources (Fireflies.ai API, folder watcher, manual upload)
- **Auto-links** mentions to existing notes in your vault
- **Generates AI summaries** with action items and decisions (Pro)
- **Creates participant notes** automatically

## Already Using the Free Version?

If you upgrade to Pro, you can easily add AI enrichment to your existing meeting notes:

1. [Get a Pro license](https://tumbucon.gumroad.com/l/meetingmind-pro)
2. [Activate your license](/pro/license) and [add your API key](/pro/api-keys)
3. Run command: `MeetingMind: Reprocess all meetings (add AI to existing notes)`

This will retroactively add summaries, action items, and AI insights to all your existing meetings.

## Requirements

- Obsidian 1.4.0 or later
- Desktop only (macOS, Windows, Linux)
- For Pro: Claude or OpenAI API key
