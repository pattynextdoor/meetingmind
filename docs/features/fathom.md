# Fathom Support

Import transcripts from Fathom using their export feature.

## How It Works

Fathom records and transcribes your meetings automatically. Export your transcripts and drop them in your watched folder for MeetingMind to process.

## Export Steps

1. **Go to [Fathom](https://fathom.video)** and sign in
2. **Open the meeting** you want to export
3. **Click "Share" or the export menu**
4. **Choose "Download transcript"**
5. **Save to your watch folder**

MeetingMind will automatically import and process it within seconds.

## Supported Export Formats

| Format | Extension | Features |
|--------|-----------|----------|
| Text | `.txt` | Speaker labels, clean formatting |
| Markdown | `.md` | Rename to `.txt` for import |

## Tips

- **Rename `.md` to `.txt`** if you download the Markdown version
- **Free tier**: Fathom offers generous free transcription
- **Batch export**: Download multiple meetings at once
- **Reprocess**: Use "MeetingMind: Reprocess current note" to add AI enrichment

## Why Use MeetingMind with Fathom?

Fathom provides great transcripts and summaries. MeetingMind adds:

- **Auto-linking** to your existing Obsidian notes
- **Participant tracking** across all your meetings
- **Vault integration** with Dataview queries
- **Custom AI processing** with your own API keys
- **Unified format** for all meeting sources in one place
