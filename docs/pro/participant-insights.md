# AI Participant Insights

Get AI-generated insights for each meeting participant.

## What You Get

In participant notes:

```markdown
## From Recent Meetings

### 2024-12-18 Project Kickoff
- **Role**: Technical Lead
- **Key Contributions**: Led API architecture discussion, proposed phased rollout
- **Action Items**: Draft technical spec, coordinate with platform team
- **Sentiment**: Engaged, solution-oriented
```

## How It Works

1. MeetingMind analyzes each participant's contributions
2. Identifies their role in the discussion
3. Extracts their key points and commitments
4. Adds insights to their participant note

## Requirements

- Pro license
- "Auto-create participant notes" enabled
- AI Enrichment enabled

## Configuration

Participant insights are automatically included when:
- **Settings → Participants → Auto-create participant notes** is ON
- **Settings → AI Enrichment** is enabled

## Benefits

- **Track contributions**: See what each person brings to meetings
- **Context for 1:1s**: Review someone's recent meeting activity
- **Team dynamics**: Understand who drives decisions
