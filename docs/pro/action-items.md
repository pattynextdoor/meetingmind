# Action Item Extraction

Automatically extract tasks and commitments from your meetings.

## What You Get

```markdown
## Action Items
- [ ] Draft migration timeline (@Patrick, due: Jan 20)
- [ ] Audit current API consumers (@Marcus)
- [ ] Schedule stakeholder review meeting (@Sarah)
- [ ] Update documentation for v2 changes (@Team)
```

## Features

- **Owner detection**: Identifies who's responsible
- **Due dates**: Extracts mentioned deadlines
- **Task format**: Uses Obsidian's checkbox syntax

## Configuration

**Settings → MeetingMind → AI Enrichment**

- Enable **Extract action items**

## Integration with Tasks Plugin

Action items use standard Obsidian task format, compatible with:
- Obsidian Tasks plugin
- Dataview task queries
- Any task management workflow
