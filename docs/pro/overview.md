# Pro Features Overview

MeetingMind Pro unlocks AI-powered features that transform your meeting transcripts into actionable knowledge.

## What's Included

| Feature | Description |
|---------|-------------|
| **AI Summaries** | 2-4 sentence meeting summaries |
| **Action Items** | Automatic task extraction with owners |
| **Decisions** | Key decisions made during the meeting |
| **Tag Suggestions** | Smart tags based on your vault's existing tags |
| **Participant Insights** | AI-generated insights per participant |
| **Entity Extraction** | Auto-create notes for issues, updates, and topics |
