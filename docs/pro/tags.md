# Smart Tag Suggestions

Get AI-recommended tags based on your meeting content and existing vault tags.

## What You Get

Tags are added to frontmatter:

```yaml
---
tags: [engineering, api, migration, q1-planning]
---
```

## How It Works

1. MeetingMind analyzes your transcript content
2. Checks your vault's existing tags
3. Suggests relevant tags that match your tagging style

## Configuration

**Settings → MeetingMind → AI Enrichment**

- Enable **Suggest tags**

## Benefits

- **Consistent tagging**: Uses tags you already have
- **Discoverable**: New meetings connect to existing tag queries
- **Time-saving**: No manual tagging required
